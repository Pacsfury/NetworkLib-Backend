# NetworkLib - Backend
NetworkLib backend is the server for the NetworkLib infrastructure.

---

## What does this backend do

This backend receives data and returns a value to them depending of the sent message.


## What is this for

This is for creating a easy, direct and simple way to create multiplayer games or any multiplayer software with a simple file.

## Current commands

New commands and functionalities will be constantly added. This alpha version includes:

* **SET** name, val: sets a variable
* **GET** name     : gets and returns the variable value
